## 项目安装&运行

- 克隆后安装相应库：
    ```
    npm install
    ```

- 程序的热编译&运行：
    ```
    npm run serve
    ```

- 对程序语法检查（ESLint）：
    ```
    npm run lint
    ```

- 编译和压缩文件：
    ```
    npm run build
    ```
