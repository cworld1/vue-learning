## 项目安装&运行

- 克隆后安装相应库:
    ```
    npm install
    ```
- 载入src文件: 从众多的带src的文件夹中选择需要回放的文件夹部分，重命名为 `src` 后置于根目录

- 程序的热编译&运行:
    ```
    npm run serve
    ```

- 对程序语法检查 (ESLint):
    ```
    npm run lint
    ```

- 编译和压缩文件:
    ```
    npm run build
    ```
